# RAG-Based GenAI Project

## Description
This project demonstrates a simple Retrieval-Augmented Generation (RAG) setup using GPT-2 and FAISS.

## How it works
1. Store documents in a list
2. Compute embeddings with SentenceTransformer
3. Retrieve the most relevant document using FAISS
4. Feed retrieved document + query to GPT-2 to generate answers

## How to Run
- Open the Colab notebook `rag_product.ipynb`
- Replace the query or documents as needed
- Run all cells
